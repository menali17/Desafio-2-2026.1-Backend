# Desafio DETRAN - Banco de Dados

Projeto desenvolvido para a modelagem e implementação de um banco de dados relacional para o monitoramento de infrações de trânsito do DETRAN. Como parte do segundo desafio de capacitação da **EngNet**

## Descrição

O banco de dados armazena informações sobre proprietários, veículos, modelos, categorias, tipos de infração, locais, agentes de trânsito e infrações registradas.

O projeto contém:

- DER - Diagrama Entidade-Relacionamento
- DLD - Diagrama Lógico de Dados
- Script físico de criação do banco
- Script de população das tabelas
- Script de consultas SQL
